## 🚀 Getting Started

### 📦 Installation

Ensure you have **Node.js (>=16)** installed, then run:

```sh
npm install
```

### ▶ Running Tests

- **Run all tests:**
  ```sh
  npx cypress run
  ```
- **Run tests in interactive mode:**
  ```sh
  npx cypress open
  ```
